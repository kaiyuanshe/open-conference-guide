# 总控进度表

## 使用说明

总控进度表由总体组维护，每周例会前更新。每一项任务都必须有负责人、截止时间、交付物和状态。

## 状态定义

| 状态 | 含义 |
| --- | --- |
| 未开始 | 已确认要做，但尚未启动 |
| 进行中 | 已有人负责并持续推进 |
| 待确认 | 需要外部人员、供应商、赞助商或总体组确认 |
| 有风险 | 可能影响时间、预算、质量或现场执行 |
| 已完成 | 交付物已经产出并通过确认 |
| 取消 | 经确认不再执行 |

## 每周例会检查

1. 本周新增了哪些风险？
1. 哪些任务延期？
1. 哪些事项需要总体组决策？
1. 哪些事项需要跨组协作？
1. 下周必须完成哪三件事？
